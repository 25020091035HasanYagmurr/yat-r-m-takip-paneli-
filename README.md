Yatırım Takip Paneli (Investment Tracking Panel) 🚀
Modern arayüz tasarımı ve kullanıcı deneyimi (UI/UX) odaklı, yapay zeka destekli yatırım takip paneli prototipi.

📌 Proje Hakkında
Bu proje, yazılım geliştirme süreçlerinde yapay zekanın sadece bir kod yazıcı değil, aynı zamanda bir tasarım ve fikir ortağı olarak nasıl konumlandırılabileceğini test etmek amacıyla geliştirilmiştir. Geliştirme sürecinde Cursor kullanılarak prototipleme hızı maksimize edilmiş, temiz ve kullanıcı odaklı bir finansal arayüz ortaya çıkarılmıştır.

🛠️ Kullanılan Teknolojiler ve Araçlar
Frontend: HTML5, CSS3
Yapay Zeka Asistanı: Cursor Editor
🚀 Gelecek Planları (Yol Haritası)
Kahramanmaraş İstiklal Üniversitesi Yazılım Mühendisliği bölümündeki eğitimim doğrultusunda geliştirdiğim bu UI/UX tasarımı aslında daha büyük bir projenin görsel temelini oluşturuyor. Önümüzdeki yaz tatili boyunca Python ve API entegrasyonları üzerinde çalışarak, buradaki tasarım mantığını gerçek zamanlı finansal veriler çeken, dinamik bir Telegram yatırım takip botuna dönüştürmeyi hedefliyorum.
